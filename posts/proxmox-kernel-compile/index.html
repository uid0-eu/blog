Proxmox compile kernel

August 30, 2018 Eine Minute Lesezeit
Linux

Requirements:

apt-get install build-essential fakeroot bison flex
apt-get build-dep linux
apt-get install asciidoc --no-install-recommends

Kernel Quellen

git clone --depth 1 git://git.proxmox.com/git/pve-kernel.git
make

Deb installieren

Hat alles geklappt findet man folgende deb Dateien:

linux-tools-4.4_4.4.40-80_amd64.deb
proxmox-ve_4.4-80_all.deb
pve-firmware_1.1-10_all.deb
pve-headers-4.4.40-1-pve_4.4.40-80_amd64.deb
pve-headers_4.4-80_all.deb
pve-kernel-4.4.40-1-pve_4.4.40-80_amd64.deb
